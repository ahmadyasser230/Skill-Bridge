# دليل البدء السريع - منصة طلاب غزة

## الخطوات السريعة للتشغيل

### 1️⃣ تثبيت المتطلبات الأساسية

تأكد من تثبيت:
- Node.js (الإصدار 14 أو أحدث)
- MongoDB

### 2️⃣ تشغيل MongoDB

افتح نافذة Terminal وشغل MongoDB:
```bash
mongod
```
اترك هذه النافذة مفتوحة.

### 3️⃣ تشغيل Backend

افتح نافذة Terminal جديدة:
```bash
cd backend
npm install
cp .env.example .env
npm start
```

يجب أن ترى: "الخادم يعمل على المنفذ 5000"

### 4️⃣ تشغيل Frontend

افتح نافذة Terminal ثالثة:
```bash
cd frontend
npm install
npm start
```

سيفتح المتصفح تلقائياً على http://localhost:3000

### 5️⃣ البدء في الاستخدام

1. سجل حساب جديد
2. اذهب إلى لوحة التحكم
3. جرب طلب مساعدة أو تصفح الدورات!

## الحسابات التجريبية

للاختبار، يمكنك إنشاء حسابين:
- حساب 1: طالب يطلب مساعدة
- حساب 2: طالب يقدم مساعدة

## المشاكل الشائعة

**مشكلة**: MongoDB لا يعمل
**الحل**: تأكد من تثبيت MongoDB وتشغيله باستخدام `mongod`

**مشكلة**: Backend لا يتصل بقاعدة البيانات
**الحل**: تحقق من ملف `.env` وتأكد من صحة `MONGODB_URI`

**مشكلة**: Frontend لا يتصل بـ Backend
**الحل**: تأكد من أن Backend يعمل على المنفذ 5000

## نصائح للتطوير

- استخدم `npm run dev` في Backend للتطوير مع إعادة التشغيل التلقائي
- افتح MongoDB Compass لعرض قاعدة البيانات بصرياً
- استخدم Postman لاختبار API endpoints
